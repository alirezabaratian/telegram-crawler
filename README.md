# telegram-crawler

This project is for crawling messages of Telegram channels using Telegram client API 

## Use

0. Clone the repo
1. Generate your client credentials on [Telegram](https://my.telegram.org)
2. Define your Credentials inside main.py
3. Install dependencies using requirements.txt (preferably using virtual enviroment)
4. Run main.py
5. Authenticate the client using your Telegram phone number, the code sent by Telegram (and your 2FA password if you have one)
6. Wait for the code to finish crawling
